Understanding Sealmetrics API Rate Limits

Sealmetrics API employs a request-based rate limiting system designed for high-volume bulk data processing rather than strict query-frequency throttling.

Overview​

Sealmetrics’ API supports fast, large-scale data extraction by combining:

  • 60 requests per minute
  • Up to 10,000 records per request
  • Burst capability (all 60 requests can be executed simultaneously)
  • 600,000 records per minute throughput

This makes Sealmetrics one of the highest‑capacity analytics APIs in the industry.

Batch Processing Pattern (JavaScript)​

const batchSize = 10000;
const totalBatches = 60;
const requestInterval = 1000; // 1 sec

async function fetchAllData() {
  const allRecords = [];

  for (let i = 0; i < totalBatches; i++) {
    const offset = i * batchSize;

    const response = await fetch(`/api/v1/analytics/events`, {
      method: 'GET',
      headers: { 'Authorization': 'Bearer your_api_token' },
      params: {
        limit: batchSize,
        offset,
        start_date: '2025-01-01',
        end_date: '2025-01-31'
      }
    });

    const batch = await response.json();
    allRecords.push(...batch.data);

    if (i < totalBatches - 1) {
      await new Promise(resolve => setTimeout(resolve, requestInterval));
    }
  }

  return allRecords;
}

Retry Logic (JavaScript)​

async function makeRequestWithRetry(url, options = {}, maxRetries = 3) {
  for (let attempt = 1; attempt <= maxRetries; attempt++) {
    try {
      const response = await fetch(url, options);

      if (response.status === 429) {
        const retryAfter = parseInt(response.headers.get('Retry-After') || '60');
        if (attempt < maxRetries) {
          await new Promise(res => setTimeout(res, retryAfter * 1000));
          continue;
        }
        throw new Error('Max retries reached');
      }

      if (!response.ok) {
        throw new Error(`HTTP ${response.status}`);
      }

      return await response.json();
    } catch (error) {
      if (attempt === maxRetries) throw error;
      await new Promise(res => setTimeout(res, Math.pow(2, attempt) * 1000));
    }
  }
}
