Feature Deep Dive​

500,000 Monthly Pageviews​

Your pageview allowance includes:

  • All tracked page loads
  • Virtual pageviews (SPAs)
  • Custom events do NOT count against pageviews

What counts:

Page load → 1 pageview
Virtual navigation (SPA) → 1 pageview
Event (add-to-cart) → 0 pageviews
Conversion → 0 pageviews

Basic API Access​

Research plan API access includes:

Available:

  • Read reports data
  • Export data (CSV format)
  • Basic rate limits (30 req/min)

Not available:

  • Write operations
  • Real-time data endpoints
  • Advanced filtering
  • Bulk exports

When to Upgrade​

Consider upgrading to Guard when:

  • You manage multiple websites
  • Traffic exceeds 500K pageviews/month
  • You need more than 2 team members
  • You want real-time anomaly alerts
  • You need BigQuery integration

Getting Started​

  1. Sign up at app.sealmetrics.com/register
  2. Start your 14-day free trial
  3. Install the tracking pixel
  4. Choose Research plan before trial ends

No credit card required to start your trial.

Upgrading​

Upgrade to Guard or Growth anytime:

  1. Go to Settings → Billing
  2. Click Change Plan
  3. Select new plan
  4. Confirm upgrade

Upgrades take effect immediately. You'll be charged a prorated amount for the remainder of your billing period.
