URL: https://docs.sealmetrics.com/api/shopify

# Shopify Integration API

Endpoints used by the SealMetrics Shopify app — the per-site connection status, the install link generator, and the disconnect action. The OAuth callback, GDPR, and uninstall webhooks live under the same router but are called **by Shopify**, not by your code, and are not documented here.

**Base path:** `/shopify`

Required scope: standard session for read; `write` for disconnect.

**Info:**
Stores installed via the Shopify app go through Shopify's **Managed Pricing** flow instead of Stripe. See [Billing → Shopify Billing](./billing#shopify-billing-managed-pricing) for the three Managed Pricing endpoints.

---

## Get Connection Status

```http
GET /shopify/sites/{site_id}/status
```

Whether a site is currently connected to a Shopify shop. Used by the dashboard to render the "Connect" / "Connected to acme.myshopify.com" UI state.

**Response (connected):**

```json
{
  "success": true,
  "data": {
    "connected": true,
    "shop": "acme.myshopify.com",
    "connected_at": "2025-01-05T10:00:00Z"
  }
}
```

**Response (not connected):**

```json
{
  "success": true,
  "data": { "connected": false }
}
```

---

## Generate Install URL

```http
GET /shopify/sites/{site_id}/install-url?shop={shop}
```

Generates the Shopify OAuth install URL for a shop. Redirect the user's browser to the returned URL to start the install flow.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `shop` | string | Yes | Target Shopify shop domain (e.g. `acme.myshopify.com`) |

**Response:**

```json
{
  "success": true,
  "data": {
    "install_url": "https://acme.myshopify.com/admin/oauth/authorize?client_id=..."
  }
}
```

After install, Shopify redirects to `/shopify/callback`, which is handled server-side. The user lands back in the SealMetrics dashboard with the connection live.

---

## Disconnect Shopify

```http
DELETE /shopify/sites/{site_id}
```

**Required scope:** `write`. Disconnects the site from its Shopify shop and revokes the stored access token. Returns `204 No Content`.

---

## Endpoints called by Shopify (not by you)

These exist on the same router but are invoked by Shopify's servers — you should not call them directly:

| Endpoint | Purpose |
|----------|---------|
| `GET /shopify/callback` | OAuth callback after install. Exchanges the temporary code for an access token |
| `GET /shopify/custom-install`, `GET /shopify/custom-callback` | Custom-app variant of the install flow (used when distributing the app outside the Shopify App Store) |
| `POST /shopify/uninstalled` | Webhook fired when the merchant uninstalls the app |
| `POST /shopify/webhooks/customers-data-request` | GDPR — customer data export request |
| `POST /shopify/webhooks/customers-redact` | GDPR — customer data deletion |
| `POST /shopify/webhooks/shop-redact` | GDPR — shop data deletion after 48h |

`POST /shopify/custom-apps` is a **superadmin-only** endpoint used to register new custom-app credentials.
