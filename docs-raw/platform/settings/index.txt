# Settings Overview

The Settings area lets you configure every aspect of your Sealmetrics account. Access it by clicking the gear icon in the main navigation.

## Settings Categories

### Account Settings

Configure your account and organization.

| Setting | Description |
|---------|-------------|
| [General](./account/general) | Account name, timezone, default date range |
| [Users & Teams](./account/users) | Manage team members and permissions |

### Tracking Settings

Configure how data is collected.

| Setting | Description |
|---------|-------------|
| [Pixel Builder](./tracking/pixel-builder) | Generate and customize your tracking code |
| [Content Grouping](./tracking/content-grouping) | Group pages into content categories |
| [Custom Dimensions](./tracking/custom-dimensions) | Define custom data fields |

### Integrations

Connect external services.

| Setting | Description |
|---------|-------------|
| [BigQuery](./integrations/bigquery) | Export data to Google BigQuery |
| [Webhooks](./integrations/webhooks) | Send data to external endpoints |

### Advanced

Power user settings.

| Setting | Description |
|---------|-------------|
| [Audit Log](./advanced/audit-log) | View account activity history |
| [LLM Settings](./llm) | Configure AI model for LENS insights |

## Quick Access

## Settings by Role

Access to settings depends on your organization role and site-level role:

| Role | Can Access |
|------|------------|
| **Owner** | All settings, including billing and organization management |
| **Admin** | Tracking, integrations, member management (no billing) |
| **Member (Editor)** | Site-specific settings for assigned sites only |
| **Member (Viewer)** | View settings only, cannot modify |

## Common Tasks

### Set Up Tracking

1. Go to **Settings → Tracking → Pixel Builder**
2. Configure your pixel options
3. Copy the generated code
4. Install on your website

[Pixel Builder guide →](./tracking/pixel-builder)

### Add Team Members

1. Go to **Settings → Organization → Members**
2. Click **Add Member**
3. Enter email and select role
4. User receives invitation email

[User management guide →](./account/users)

### Connect BigQuery

1. Go to **Settings → Integrations → BigQuery**
2. Create a GCP service account
3. Upload credentials
4. Configure sync options

[BigQuery setup guide →](./integrations/bigquery)

### View Activity History

1. Go to **Settings → Advanced → Audit Log**
2. Browse or search activity
3. Filter by user, action, or date

[Audit log guide →](./advanced/audit-log)
