# Introduction to SealMetrics API

Learn the basics of the SealMetrics API and how to integrate privacy-first analytics into your applications. This section introduces the core concepts, authentication methods, and capabilities of our REST API.

The SealMetrics API enables developers to programmatically access analytics data, send custom events, track conversions, and build custom integrations. Whether you're building a mobile app, creating backend tracking, or developing custom dashboards, understanding these fundamentals will help you leverage the full power of consentless analytics in your projects.
