## Feature Deep Dive

### 3 Accounts (Properties)

Manage up to 3 separate tracking properties:

```
Account 1: main-store.com
Account 2: blog.company.com
Account 3: uk.main-store.com
```

Each account has:
- Separate tracking pixel
- Independent data
- Own anomaly rules
- Shared user access

## When Guard is Right

✅ **Choose Guard if:**
- You have 2-3 websites to track
- Traffic is between 500K - 2M pageviews
- You need real-time alerts
- BigQuery integration is important
- You want faster support response

❌ **Consider Growth instead if:**
- You have more than 3 websites
- Traffic exceeds 2M pageviews
- You need unlimited team members
- You want dedicated support
- You need custom integrations

When you upgrade:

1. All your data is preserved
2. Additional accounts become available immediately
3. All 60+ anomaly rules activate
4. BigQuery integration unlocks
5. API limits increase

**Prorated billing:** You'll only pay the difference for the remaining billing period.
