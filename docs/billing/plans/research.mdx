## Feature Deep Dive

### 500,000 Monthly Pageviews

Your pageview allowance includes:
- All tracked page loads
- Virtual pageviews (SPAs)
- Custom events do NOT count against pageviews

**What counts:**
```
Page load → 1 pageview
Virtual navigation (SPA) → 1 pageview
Event (add-to-cart) → 0 pageviews
Conversion → 0 pageviews
```

### 10 Basic Anomaly Detection Rules

The Research plan includes these detection rules:

| Rule | Description |
|------|-------------|
| Traffic drop | Alert on >40% traffic decrease |
| Traffic spike | Alert on >60% traffic increase |
| Conversion drop | Alert on >30% conversion decrease |
| Revenue drop | Alert on >35% revenue decrease |
| Bounce rate spike | Alert on >30% bounce rate increase |
| Pixel errors | Alert on tracking issues |
| Zero traffic | Alert when no data received |
| Source change | Alert on traffic source shifts |
| Device shift | Alert on device distribution changes |
| Geographic change | Alert on country distribution changes |

**Not available:**
