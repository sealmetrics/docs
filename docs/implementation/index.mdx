# Implementation Guide

Learn how to implement SealMetrics tracking on your website. This section provides comprehensive guides for setting up privacy-first analytics across different use cases and technologies.

Whether you're running an e-commerce store, a content website, or a complex single-page application, you'll find step-by-step instructions to integrate SealMetrics without requiring user consent. Our implementation guides cover everything from basic page view tracking to advanced conversion tracking and custom event monitoring.
