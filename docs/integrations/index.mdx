## Quick Start

### Google Tag Manager

1. Download the template: [template.tpl](https://github.com/sealmetrics/gtm-template/raw/main/template.tpl)

2. In GTM, go to **Templates** → **New** → **Import** the file

3. Create a new tag using the **Sealmetrics Tracking** template

4. Configure your **Account ID** and event type

5. Add triggers and publish your container

[Get started with GTM Template →](./google-tag-manager)

### MCP Server for Claude

1. Install the server (choose one method):
   ```bash
   # Recommended: using pipx (isolated environment)
   pipx install sealmetrics-mcp

   # Alternative: using pip
   pip install sealmetrics-mcp

   # Alternative: using uvx
   uvx sealmetrics-mcp
   ```

2. Configure Claude Desktop with your [API token](/api/authentication/how-to-obtain-api-token) and [Account ID](/api/core-api-endpoints/account-endpoints)

3. Restart Claude Desktop completely

4. Start asking questions about your analytics data

[Get started with MCP Server →](./mcp-server)
