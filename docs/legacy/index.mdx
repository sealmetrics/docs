# Legacy Documentation

This section contains documentation for **Sealmetrics v1**, which is being phased out in favor of the new Sealmetrics v2 platform.

:::warning
The documentation in this section refers to the legacy version of Sealmetrics. For new implementations, please refer to the main documentation sections which cover Sealmetrics v2.
:::

## What's included here

- **[API v1.0](./v1/api/)** - REST API documentation for the legacy platform
- **[Plugins & Modules v1.0](./v1/plugins-and-modules/)** - WordPress, WooCommerce, Magento, PrestaShop, and other platform integrations
- **[Platform Internals](./v1/platform-internals/)** - Technical details about how the legacy system processes and stores data

## Migration to v2

If you're currently using Sealmetrics v1 and need to migrate to v2, please contact our support team for assistance with the transition.
