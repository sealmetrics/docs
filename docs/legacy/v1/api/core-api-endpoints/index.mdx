# Core API Endpoints

Access aggregated analytics data through these REST API endpoints. This section documents all available endpoints for retrieving analytics metrics, building custom dashboards, and integrating Sealmetrics data into your applications.

The Core API provides programmatic access to your analytics data, including page performance, conversion funnels, ROAS calculations, and account information. Use these endpoints to create custom reports, automate data exports, build internal dashboards, or integrate analytics into your existing business intelligence tools.

## Available Endpoints

- [API Endpoints Overview](./api-endpoints) - Complete reference of all available endpoints and their capabilities
- [Account Endpoints](./account-endpoints) - Retrieve account information and configuration data
- [Funnel Endpoint](./funnel-endpoint) - Access conversion funnel data and drop-off analysis
- [Pages Endpoint](./pages-endpoint) - Get page performance metrics and traffic data
- [ROAS Evolution Endpoint](./roas-evolution-endpoint) - Track Return on Ad Spend over time
- [Set Click Endpoint](./set-click-endpoint) - Record click events for attribution tracking
