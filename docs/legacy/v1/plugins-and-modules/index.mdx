## E-commerce Integrations

Complete e-commerce tracking for online stores. Track the entire customer journey from product discovery to purchase.

### What's Tracked

| Event | Type | Description |
|-------|------|-------------|
| `pageview` | Pageview | Every page load |
| `product_view` | Microconversion | Product detail page visits |
| `add-to-cart` | Microconversion | Items added to shopping cart |
| `checkout1` | Microconversion | Checkout initiated |
| `checkout2` | Microconversion | Customer info entered |
| `checkout3` | Microconversion | Payment method selected |
| `purchase` | Conversion | Order completed |

## Common Features

All integrations share these features:

### Privacy First

- ‚ùå No order IDs or transaction IDs stored
- ‚ùå No email addresses collected
- ‚ùå No personal identifiable information (PII)
- ‚úÖ GDPR compliant by design
- ‚úÖ No cookies required

### Consistent Architecture

- **Single script load** - Sealmetrics JS loaded once (async)
- **Global event queue** - `window.sealmetricsTrack` array
- **Pageview first** - Always fires before other events
- **Debug mode** - Console logging for troubleshooting

### Attribute Normalization

Product attributes are automatically normalized for cross-platform analytics:

| Original Values | Normalized To |
|----------------|---------------|
| color, colour, colore, couleur, farbe | `colour` |
| size, talla, taille, gr√∂√üe, grosse | `size` |
| material, materiale, mati√®re | `material` |
| weight, peso, poids | `weight` |
