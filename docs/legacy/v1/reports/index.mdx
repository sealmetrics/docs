# Reports (Legacy v1)

This section contains legacy documentation for Sealmetrics v1 reports. These reports have been replaced by the new v2 reporting system.

## Deprecated Reports

The following v1 reports are no longer available in the current platform:

| v1 Report | Replaced By |
|-----------|-------------|
| [Dashboard](./dashboard) | [Overview](/reports/overview) |
| [Acquisition](./acquisition) | [Sources](/reports/sources) |
| [ROAS Evolution](./roas-evolution) | [Evolution](/reports/evolution) + [Conversions](/reports/conversions) |
| [Sales Funnel](./sales-funnel) | [Funnel](/reports/funnel) |

## Migration Notes

### Dashboard → Overview

The v1 Dashboard has been replaced by the Overview report which provides:
- Real-time live visitor counter
- Enhanced metric cards with comparison deltas
- Improved chart interactions

### Acquisition → Sources

The v1 Acquisition report is now the Sources report with:
- Six tabs for UTM dimension analysis
- Enhanced filtering per tab
- Visual charts for each tab

### ROAS Evolution → Evolution + Conversions

ROAS tracking is now available through:
- Evolution report for time-based trends
- Conversions report for detailed conversion analysis
- External cost data can be combined with Revenue metrics

### Sales Funnel → Funnel

The v1 Sales Funnel has been enhanced with:
- Configurable funnel steps
- Visual drop-off indicators
- Step-by-step metrics table
