## Understanding the AI Insights

### How Insights Are Generated

1. **Data Analysis** - LENS analyzes all metrics from the week
2. **Pattern Detection** - Identifies significant changes and correlations
3. **Context Gathering** - Considers historical patterns and seasonality
4. **Insight Generation** - AI formulates actionable observations
5. **Prioritization** - Ranks insights by potential business impact

### Insight Types

| Type | Description | Example |
|------|-------------|---------|
| **Opportunity** | Potential for improvement | "Email ROAS suggests scaling" |
| **Risk** | Issues requiring attention | "Mobile conversion declining" |
| **Trend** | Directional changes | "YoY growth accelerating" |
| **Anomaly** | Unusual patterns | "Traffic spike from unknown source" |
| **Correlation** | Related changes | "Page speed affecting conversion" |

## Best Practices

### For Maximum Impact

1. **Keep recipient list focused** - Only those who will act on insights
2. **Set appropriate delivery time** - Before weekly meetings
3. **Review and refine** - Adjust metrics shown based on feedback
4. **Add context** - Use annotations for known events

### For Busy Executives

1. **Use summary-only option** - Condensed to key metrics
2. **Highlight thresholds** - Only show significant changes
3. **Include recommendations** - Actionable next steps
4. **Mobile-friendly** - Readable on phone

## Report Sections

A typical Executive Summary includes:

1. **Performance Overview** - Key metrics vs previous period
2. **Traffic Analysis** - Sources, channels, campaigns
3. **Conversion Summary** - Goals, revenue, transactions
4. **LENS Insights** - AI-detected patterns and anomalies
5. **Recommendations** - Suggested actions for improvement
