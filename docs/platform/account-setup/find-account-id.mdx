# How to Find Your Site ID

Your **Site ID** is a unique identifier assigned to each site in Sealmetrics. You need it to install the tracking pixel, configure API access, and set up integrations.

The Site ID is set when the site is created and **cannot be changed** afterwards.

---

## Steps to Find Your Site ID

### 1. Open **Organization → Sites**

In the left sidebar, expand the **Organization** group and click **Sites** to open the **Site Management** page (`/settings/sites`).

### 2. Open your site's settings

Find your site in the **Your Sites** list and click **Configure**. This opens the site's settings page (`/settings/sites/{siteId}`).

### 3. Copy the Site ID

On the **General** tab, the **Site ID** is shown next to the **Site Name** field. Use the **copy button** beside it to copy it to your clipboard.

---

You can also see the Site ID listed next to each site under **Organization → Settings** (in the organization's **Sites** card).
