# How to Delete an Account

Follow these simple steps:

<img src="/img/account-setup/delete-account.png" alt="Delete account screenshot"/>

## Steps  
1. Open the monitor menu.  
2. Go to **Accounts**.  
3. Click the **Delete** icon next to the account you want to remove.  
4. Confirm the deletion.

---

If you delete an account, all related data is permanently removed.
