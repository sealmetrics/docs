# How to Add a New User

If you need to add a teammate or grant access to a client, follow these steps.

## 1. Open the Monitor Menu → Users

![Add New User Step//img/add-user-1.png)

## 2. Click “Add User”

![Add New User Step//img/add-user-2.png)

## 3. Complete the User Details

Fill in:

- Name  
- Email  
- Assigned Accounts  
- User Role  

![Add New User Step//img/add-user-3.png)

Click **Create** to finish.

---

Users will receive an automatic invitation to access the platform.
