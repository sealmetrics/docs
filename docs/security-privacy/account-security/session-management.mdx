## Session Security Best Practices

### For Individual Users

- âœ… Sign out from shared computers
- âœ… Review sessions monthly
- âœ… Use 2FA on all sessions
- âœ… Sign out old sessions regularly
- âŒ Don't stay signed in on public computers
- âŒ Don't ignore unknown sessions

### For Admins

- âœ… Set reasonable session timeouts
- âœ… Monitor team sessions regularly
- âœ… Disable "Remember Me" for high-security accounts
- âœ… Force sign-out when employees leave
- âœ… Review sessions during security incidents

## Troubleshooting

### "Session expired unexpectedly"

- Check if admin changed timeout settings
- Verify you didn't sign out from another device
- Clear browser cookies and sign in again

### "Can't see my sessions"

- Make sure you're signed in
- Check you have permission (admins can restrict)
- Try refreshing the page

### "Old sessions still showing"

- Sessions may take a few minutes to update
- "Last active" shows last recorded activity
- Sign out to immediately remove
