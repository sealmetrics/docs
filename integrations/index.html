Quick Installation Guide​

For Most Websites​

  1. Choose your platform from the list above
  2. Download the plugin/module/script
  3. Install following the platform-specific instructions
  4. Enter your Account ID (found in your SealMetrics dashboard)
  5. Save and publish

For Custom Implementations​

If your platform isn't listed, you can:

  1. Use the GTM Container Template for tag manager-based installation
  2. Use the JavaScript Tracker directly
  3. Use the API for server-side tracking

Need Help?​

If your platform isn't listed or you need assistance, contact us at support@sealmetrics.com.
