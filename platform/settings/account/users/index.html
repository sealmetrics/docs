Users & Teams

Manage who has access to your Sealmetrics account and what they can do.

Accessing User Settings​

  1. Click the gear icon in the main navigation
  2. Select Account → Users

Invitation Process​

  1. User receives email invitation
  2. They click "Accept Invitation"
  3. Create password (or use SSO)
  4. Immediately gains access

Custom Roles (Growth Plan)​

Create custom roles with specific permissions:

  1. Go to Account → Users → Roles
  2. Click Create Custom Role
  3. Name the role and select permissions
  4. Assign to users

  2. Modify settings:

Change User Role​

  1. Find user in the list
  2. Click role dropdown
  3. Select new role
  4. Confirm change

Role changes take effect immediately.

Remove User​

  1. Find user in the list
  2. Click ⋮ → Remove
  3. Confirm removal
Remove User

• All dashboards and reports
• LENS AI
