Web Analytics Prompts

A curated library of production-ready prompts for the SealMetrics MCP server and cross-platform combinations with Google Ads MCP, Meta Ads MCP, and Google Search Console MCP.

Every prompt in this section is designed for two ideal customer profiles:

  • Sales & Direct Directors of hotels (independent and chains).
  • CMOs and Ecommerce Managers of online stores.

How to use these prompts​

  1. Open Claude Desktop, Claude Code, or any MCP-compatible client.
  2. Make sure the SealMetrics MCP server is connected. For multi-MCP prompts, also connect the corresponding platform MCP (Google Ads, Meta Ads, GSC).
  3. Pick a prompt from the category that matches your goal.
  4. Click the copy button on the prompt box, paste it into your chat, and replace the placeholders (e.g. {site_id}, {date_range}).
  5. Run it. Iterate from there with follow-up questions — the agent will keep the context.

Prompt anatomy​

Every prompt follows the same structure so you can adapt it to your own workflows:

  • Context — what the agent should know before answering.
  • Instruction — the actual task.
  • Parameters — variables you should adjust.
  • Output format — how you want the answer back.

Categories​

  1. Revenue Leak Detection — find invisible money lost to bots, broken pages, and weak conversion paths.
  2. Conversion Optimization — funnel analysis, micro-to-macro ratios, device gaps.
  3. Traffic & Quality Control — anomaly detection, bot share, suspicious sessions.
  4. Attribution & Channels — real channel performance, cannibalization, last-click ROAS.
  5. Geography & Segmentation — country revenue, expansion opportunities, geo-level CR.
  6. Spend Efficiency — pause underperformers, surface waste, reduce CAC.
  7. Alerts & Monitoring — daily anomaly digests and threshold-based alerts.
  8. Executive Reporting — C-level digests, QoQ KPIs, silent winners.
  9. Hotels — direct booking funnel, OTA shift, country mix.
  10. Ecommerce — cart-to-purchase, AOV by channel, returning customer split.
  11. Product Properties — SKU, size, color, room type, length-of-stay analysis.
  12. Forecasting — month-end, quarter, year, Black Friday, hotel pickup curve.
  13. SealMetrics + Google Ads — real CPA, wasted spend, geo bid optimization.
  14. SealMetrics + Meta Ads — iOS attribution gap, audience ROAS, creative fatigue.
  15. SealMetrics + Search Console — query CR, ranking-to-revenue, cannibalization.
  16. Cross-platform — full-funnel ROAS across paid + organic + direct.

Stable prompt IDs​

Every prompt has a stable ID like SEAL-001. Use it to link from internal docs, sales decks, or LinkedIn posts.
